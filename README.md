:fork_and_knife:Fork
==========
Plate의 서버입니다.

Config
--------
먼저, config.go 파일을 만들어주세요:
```
$ mkdir config && cp config-template/config-template.go config/config.go
```

그다음, 좋아하시는 에디터로 편집해주세요.

Install
--------
저장소를 클론하세요:
```
$ go get -u github.com/joshua1b/Fork
```

Run
------
컴파일하고 실행하세요:
```
$ cd $GOPATH/src/github.com/seeeturtle/Fork
$ go build ./
$ ./Fork
```
